# Gra Arkanoid

Prosta gra typu Arkanoid napisana w C++ z wykorzystaniem biblioteki SFML.  
Projekt zawiera podstawową fizykę, kolizje, sterowanie paletką, ruch piłki oraz niszczalne bloki.

## Funkcje

- klasyczna rozgrywka Arkanoid
- sterowanie paletką przez gracza
- odbijająca się piłka z detekcją kolizji
- niszczalne bloki z punktami życia
- czytelna, obiektowa struktura kodu
- renderowanie oparte na SFML

## Wymagania

Do uruchomienia lub kompilacji potrzebujesz:

- C++17 lub nowszego
- SFML 3.0 lub nowszego
